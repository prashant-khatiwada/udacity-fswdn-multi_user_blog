    Multi-User-Blog

    Multi-User-Blog is a project submission for the Udacity Full Stack Web Developer Nanodegree program.

    Setup

    You can run this project with via cloning from GitHub and running dev_appserver.py in the project's root directory. This will run a local instance of the multi-user-blog and can be viewed localhost:8080.

    GitHub

    Available on Github. Clone the project using git clone command (or fork the project)

    		# Clone the repository directly 
		$ git clone https://github.com/quirkyPrashant/multi-user-blog.udacity.fswdn
