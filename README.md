# multi-user-blog.udacity.fswdn
Multi User Blog created as a learning experience for Full Stack Web Dev Nanodegree

A simple blog allowing for multiple Users to post their content

To run the application locally:

Clone the repository from Github
1. Install Google App Engine for Python by following instructions
2. Open up your terminal and navigate to your project folder.
3. Type "dev_appserver.py ." to start the server locally.
4. Open a web browser and enter in the URL localhost:8080/blog.
